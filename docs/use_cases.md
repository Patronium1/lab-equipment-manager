 Прецеденты использования

 1. Подача заявки на оборудование
Актор: Преподаватель
Описание: Преподаватель подает заявку на использование оборудования
Основной поток:
1. Вход в систему
2. Переход на вкладку "Новая заявка"
3. Выбор оборудования
4. Заполнение формы
5. Отправка заявки

 2. Рассмотрение заявки
Актор: Администратор
Описание: Администратор рассматривает и изменяет статус заявки
Основной поток:
1. Вход в систему
2. Просмотр списка заявок
3. Выбор заявки
4. Изменение статуса
5. Добавление комментария

 3. Управление оборудованием
Актор: Администратор
Описание: Добавление и редактирование оборудования
Основной поток:
1. Вход в систему
2. Переход на вкладку "Оборудование"
3. Добавление/редактирование
4. Сохранение изменений
