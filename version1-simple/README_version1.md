Версия 1.0 - Базовая

Функционал
- Аутентификация пользователей
- Ролевой доступ (администратор/преподаватель)
- Подача заявок на оборудование
- Просмотр заявок
- Изменение статуса заявок (для администратора)

Особенности
- Минималистичный интерфейс
- Английские статусы
- Базовая цветовая подсветка
- Отсутствие гостевого доступа

Структура БД
- Таблица пользователей (users)
- Таблица оборудования (equipment)
- Таблица заявок (requests)
