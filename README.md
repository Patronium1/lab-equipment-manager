 LabEquipment Manager

Система управления лабораторным оборудованием для учебных заведений.

 О проекте
Программа автоматизирует процесс учета и бронирования лабораторного оборудования. 
Предоставляет интерфейсы для трех типов пользователей:
- Администратор: полный доступ ко всем функциям
- Преподаватель: подача заявок на оборудование
- Гость: просмотр без возможности редактирования
